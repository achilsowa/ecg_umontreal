********************DESCRIPTION********************
 Simple static webserver where the user can a patient and plot the patient's ecg

********************REQUIREMENTS******************** 
- python3. All the lib used are provided directly by python3 core. 
- being online while running it. We rely on bootstrapp for styling, so you need to be online to access the bootstrap stylesheet. 

********************HOW TO RUN********************
You'll need python3, on the cli 
_> python3 server.py
